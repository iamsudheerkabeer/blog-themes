# Premium Blogger Theme 🚀

A highly optimized, feature-rich, and dynamic Blogger template designed for modern news portals, content creators, and serial novel publishers. Developed with clean code architecture and advanced JavaScript logic.

## ✨ Key Features

- 🌓 **Dark/Light Mode:** Smooth toggling with local storage persistence.
- ⚙️ **Centralized `SiteConfig`:** Easily customize colors, logos, ads, and navigation without hunting through complex HTML code.
- 📰 **Dynamic News Ticker & Trending Sidebar:** Auto-fetches latest posts from Blogger feeds smoothly.
- 💰 **Smart Ad Injection:** Dynamically injects Ads and "Read Also" related posts smartly between paragraphs for maximum CTR.
- 📖 **Auto Chapter Navigation:** Automatically generates Next/Previous and Chapter Number pagination for serialized content (e.g., Novels/Stories).
- 🛡️ **Anti-Theft Protection:** Includes optional obfuscated domain-lock scripts to prevent unauthorized theme stealing.
- 📋 **Enhanced Copy Function:** User-friendly copy-to-clipboard functionality with toast notifications.

## 📂 Repository Structure

- `main-theme.xml` & `Theme version 21.xml` - Complete Blogger XML theme files.
- `configdata.js` - The plain-text configuration object for easy customization.
- `main original.js` - The core logic for ads, feeds, and UI rendering (Open for learning/editing).
- `copy etc.js` - Dedicated script for advanced chapter navigation and copy functionalities.
- `vazhchayugam main obfuscator.js` - The secured/encrypted version of the core logic with domain-lock.

## 🛠️ How to Use

1. Copy the code from `main-theme.xml` and paste it into your Blogger Theme's HTML editor.
2. Search for the `SiteConfig` object in the script section.
3. Update the `logoUrl`, `social` links, `categories`, and `ads` with your own details.
4. Save the theme and you are good to go!
